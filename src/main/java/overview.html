DebugTrace class has some logging methods.

DebugTrace クラスには、ログ出力用のメソッドがあります。

The following are a sample source used DebugTrace methods and a log of when the sample was executed.
Log bodys are automatically indent.
以下は、DebugTrace のメソッドを使用したサンプルソースとそれを実行した際のログです。
ログの内容は、自動的にインデントされます。
Sample source:
サンプルソース:
package sample;

import org.debugtrace.DebugTrace;

public class Sample2 {
  // main
  public static void main(String[] args) {
    DebugTrace.enter();
    DebugTrace.print("args", args);

    sub(args);

    DebugTrace.leave();
  }

  // sub
  private static void sub(String[] args) {
    DebugTrace.enter();

    DebugTrace.print("args", args);

    DebugTrace.leave();
  }
}

DebugTrace read DebugTrace.properties file in the classpath on startup.
You can specify following properties in the DebugTrace.properties file.
DebugTrace は、開始時のクラスパスにある DebugTrace.properties を読み込みます。
DebugTrace.properties ファイルには、以下のプロパティを指定できます。

Properties of DebugTrace.properties file:
DebugTrace.properties のプロパティ一覧:
NameValueDefault
logger Logger DebugTrace uses
Select from follows.
    Jdk : When you use JDK Logger
    Log4j : When you use Log4J 1
    Log4j2 : When you use Log4J 2
    SLF4J : When you use SLF4J
    Std$Out : When you output the standard output
    Std$Err : When you output the standard error output
 DebugTrace が使用するロガー
以下から選択してください。
    Jdk : JDK Logger を使用
    Log4j : Log4J 1 を使用
    Log4j2 : Log4J 2 を使用
    SLF4J : SLF4J を使用
    Std$Out : 標準出力に出力
    Std$Err : 標準エラー出力に出力
 Std$Out
logLevel Log level DebugTrace outputs
Select from follows.
    default : The lowest level for each logger
    finest, finer, fine, config, info, warning, severe : JDK Logger
    trace, debug, info, warn, error, fatal : Log4J 1 or 2
    trace, debug, info, warn, error : SLF4J
 DebugTrace が出力するログレベル
以下から選択してください。
    default : それぞれのロガーの最低レベル
    finest, finer, fine, config, info, warning, severe : JDK Logger
    trace, debug, info, warn, error, fatal : Log4J 1 か 2
    trace, debug, info, warn, error : SLF4J
 default (finest: JDK Logger, trace: Log4J 1, 2 or SLF4J)

    %1: The class name of the caller

    %3: The file name of the caller
